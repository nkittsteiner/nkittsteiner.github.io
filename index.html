HN Code Trends

Based on HN data

TL;DR

Using mostly all the stories published in the site news.ycombinator.com I was grouping and filtering all stories related to programming languages. The core idea is to know which programming languages are the most discussed and have more relevance in the HN ecosystem.

Disclaimer: This is a very rough estimate. In special for 'Go', 'C' and 'R' that are difficult to separate from context. And also I'm mostly a backend developer so if you thing something is wrong with this web or can be changed, feel free to comment about it!

Motivation

One of my interests was to know what programming languages are the more used, mainly by my background experience. I was a .NET developer for some years and I was completely abstracted from other languages and technologies (except Javascript of course). But I needed to get out of the bubble, so I jumped out and I made the decision to completely switch platforms. Now I work in an R + D in which I was able to learn many tools mainly free code which I'm very happy to be learning and using.

One of the questions that many niche developers have, is to know how relevant are the programming languages that they use in their developments. Everybody wants to be up to-date with latest trends. One of my favorite sources of information is without a doubt, Hacker News. I'm reading this site everyday for almost two years, and it's very cool the job that they do for keep a huge community of hackers and entrepreneurs updated.
